# garden-log
A personal gardening dashboard for tracking fruit tree health, maintenance schedules, and growth journals using JavaScript, JSON, and LocalStorage.

# GardenLog - Personal Gardening Dashboard
[cite_start]GardenLog is a specialized web application designed for home gardeners to manage the health and schedules of their fruit trees and flowers. [cite: 1, 5]

## Project Overview
[cite_start]I am building this to solve the personal challenge of tracking different care needs for plants like Meyer lemons, figs, and berries. [cite: 3, 4] [cite_start]The app features a visual dashboard, maintenance scheduling, and a growth journal to ensure every plant gets the attention it needs. [cite: 5]

## Key Features
- [cite_start]**Plant Profile Creation:** Register individual plants with nicknames and planting dates. [cite: 9, 10]
- [cite_start]**Maintenance Scheduling:** Logic-based reminders for watering and fertilizing. [cite: 11, 12]
- [cite_start]**Growth Journal:** Log milestones like budding or harvests. [cite: 13, 14]
- [cite_start]**Task Checklists:** Automated to-do lists based on plant needs. [cite: 17, 18]
- [cite_start]**Dynamic Image Fetching:** High-quality visuals integrated via Unsplash API. [cite: 28]
- [cite_start]**Local Data Persistence:** All data is saved in the browser's LocalStorage. [cite: 23, 24]

## Technical Details
- [cite_start]**Data Source:** A custom JSON encyclopedia with 10+ attributes per plant species (scientific name, ideal pH, pruning season, etc.). [cite: 26, 27]
- [cite_start]**Technologies:** HTML5, CSS3 (Sage Green & Terracotta palette), and Vanilla JavaScript (ES Modules). [cite: 30-34, 36]
